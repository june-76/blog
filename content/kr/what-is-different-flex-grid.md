### flex
백지에서 그리는 느낌(?)

### grid
모눈지에 퍼스자를 대고 그리는 느낌(?)


슬랙 의견을 종합하면
비교적 간단한 화면은 flex가 더 편리하고(실제로 나도 flex가 더 편했음)
복잡한 화면 구성이 필요할 때는 grid가 섬세하게 화면 구역을 나눌 수 있다고 한다.
근데 잘하는 사람은 굳이 안 가리고 다 잘 쓰겠지...

flex는 1차원
grid는 2차원

이미 결정된 레이아웃을 만들고 싶으면 grid 속성을 사용.
자유로운 레이아웃을 만들고 싶으면 felx 속성을 사용.
